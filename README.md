### Commit 규칙

[스터디_n주차] hejcj 과제 제출

0. repository Fork

1. Sync fork로 master branch를 동기화한다, (첨부된 사진 참조)
2. git pull origin master / 로 로컬 레포를 깃허브의 master branch와 동기화
3. 파일 수정, 추가
4. git add .
5. git commit -m 'commit message'
6. git push origin {본인-branch(main아님)} / 로 {본인 레포} {본인 branch}로 Push
7. Pull Request
